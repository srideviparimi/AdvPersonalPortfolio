News Fetching Application

The main purpose of the application is to fetch data from an API. Here I have used async/await principles for fetching the news from News API. This application has a home page and a search page. Home Page: It consists of different categories of news and click on each article to watch the full article Search Page: When we search for a query using the search-bar, It navigates to the articles that are related to the query. Pokemon Search Application

This application is used to search for the pokemons from an API. It uses Javascript asynchronous functions while fetching the data from the API and displaying it in a Pokemon Card format. To search we can use a) Name of the Pokemon b) ID of the Pokemon(1 to 1025) And displays all the information about the particular Pokemon. Note: I have used the freeCodeCamp Pokemon API to fetch the data.
